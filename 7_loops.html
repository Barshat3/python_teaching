Loops

Iteration with For & While Loops

Why Use Loops?

Loops are used to execute a block of code repeatedly until a condition is met or all items in a sequence are processed.



The two types of loops used in python are:


  • For Loops
  • While Loops

Automation

Handle repetitive tasks without writing the same line multiple times.

DRY Principle

"Don't Repeat Yourself." Clean code is maintainable code.

Data Processing

Essential for processing lists, files, and user inputs.

"Computers are good at doing the same thing over and over again. Loops are how we tell them to do it."

The For Loop

A for loop is used for iterating over a sequence (list, tuple, dictionary, set, or string).

for variable in sequence:
    # Code to be executed
  • Iterates over a sequence
  • Automatic assignment
  • Can be used to iterate over collections like lists, tuples, and dictionaries
  • Instead of sequence, you can use range() function
fruits = ["apple", "banana"]
for fruit in fruits:
    print(fruit)

# Output:
apple
banana

The range() Function

To loop through a set of code a specified number of times, we use range().

range(start, stop, step)

range(0, 5, 1) → 0, 1, 2, 3, 4

range(start, stop)

range(2, 6) → 2, 3, 4, 5

range(stop)

range(5) → 0, 1, 2, 3, 4

# Counting by 2s
for i in range(0, 6, 2):
    print(i)

# Output:
0
2
4

Nested Loops

A loop inside another loop. The "inner loop" will be executed one time for each iteration of the "outer loop".

for x in ["A", "B"]:
    for y in range(2):
        print(x, y)

Output visualization:

A 0
A 1
B 0
B 1

Commonly used for 2D grids, matrices, and combining sets of data.

The While Loop

A while loop repeats code as long as a certain boolean condition remains True.

i = 0
while i < 3:
    print(i)
    i += 1
  • Condition-based execution
  • Manual management of variables
  • Risk of infinite loops

Infinite Loops & Safety

The Problem

An infinite loop occurs when the condition never becomes False.

i = 1
while i > 0:
    print("Looping forever!")
    # i is never changed!

How to Avoid

  • Update Variables: Ensure the loop variable changes (e.g., i += 1).
  • Use Break: Add a safety break if a secondary condition is met.
  • Test Logic: Trace your condition values before running.

Loop Control

break

Stops the loop entirely.

if condition: break


for i in range(5):
    if i == 3:
        break
    print(i)

# Output:
# 0
# 1
# 2

continue

Skips current iteration.

if condition: continue


for i in range(5):
    if i == 2:
        continue
    print(i)

# Output:
# 0
# 1
# 3
# 4

pass

A null placeholder. Does nothing, used to avoid syntax errors.

for x in data:
    pass # Fill later


for i in range(3):
    if i == 1:
        pass
    print(i)

# Output:
# 0
# 1
# 2

For vs. While

Feature For Loop While Loop
Primary Use Iterating collections / fixed steps Repeating until logic changes
Control Handled by iterator Handled by manual condition
