The String Toolkit 🛠️

Strings aren't just dead text. Python gives us built-in math operators and functions to dynamically manipulate them.

Concatenation (+)

Gluing two or more text strings together back-to-back. An addition operator for text blocks!

"Py" + "thon" → "Python"

Replication (*)

Repeating a piece of text multiple times using multiplication. Must match a string with an integer!

"Hi!" * 3 → "Hi!Hi!Hi!"

Length calculation

The len() global function counts every single character inside your string, including spaces and punctuation!

len("A B") → 3
⚠️

The Static Rule: You cannot mix string math operations illegally. Trying to do "Hello" + 5 will smash your program into a screeching TypeError!
