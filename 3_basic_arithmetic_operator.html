The Math Toolbox 🛠️

Python uses standard keyboard symbols to compute actions on integers and decimals.

+

Addition

Combines values together.

5 + 3 # 8
-

Subtraction

Subtracts right from left.

10 - 4 # 6
*

Multiplication

Uses an asterisk icon symbol.

3 * 4 # 12
/

True Division

Always returns a float decimal!

7 / 2 # 3.5
⚠️

Note that standard division / always outputs a float container even if numbers divide perfectly, like 4 / 2 giving 2.0.

Anatomy of an Expression 🔍

How variables and literal constants match together seamlessly inside memory evaluation lines.

result = points * 2 + 10
Variables

result and points evaluate dynamically based on current memory box contents.

= * +

The Operators perform assignments and mathematical mutations. Spaces around them are optional but highly recommended!

2, 10

Numeric literals. Do not use quotation wrappers here or Python treats them as literal character labels instead of digits!

Advanced Math Powers 💪

Python provides three specialty symbols for powers, clean cuts, and remainders.

**

Exponentiation

Raises the left number to the power of the right number.

2 ** 3 # 2*2*2 = 8
//

Floor Division

Divides and chops off everything after the decimal point toward zero.

7 // 2 # Gives 3
%

Modulo (Remainder)

Finds the leftover remainder after dividing completely by a value.

7 % 2 # Leftover = 1
