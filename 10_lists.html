Phase 2: Anatomy & Mechanics

What is a Python List?

A Python list is an ordered, mutable sequence collection of items stored globally under a single overarching variable identifier.

Ordered Items maintain exact insertion sequences reliably.
Mutable Can be instantly modified after memory assignment.
Duplicates Identical element entries can coexist safely.

Crucial Architecture Rule: Zero-Indexing

In computing arrays and list environments, counting starts exclusively at index 0, not 1!

Index: [0] → [1] → [2]

todo_manager_starter.py Reference Boilerplate
tasks = []

while True:
    print("\n1. Add  2. View  3. Remove  4. Sort  5. Exit")
    choice = int(input("Choose option: "))

    if choice == 1:
        task = input("Enter task: ")
        tasks.append(task)
    elif choice == 2:
        for i, t in enumerate(tasks, start=1):
            print(f"{i}. {t}")
    elif choice == 3:
        num = int(input("Remove index number: "))
        tasks.pop(num - 1)
    elif choice == 4:
        tasks.sort()
    elif choice == 5:
        break
