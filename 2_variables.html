🧬 Python's 4 Core Data Types

Each type tells Python how to store and use the data. Hover a card to explore, then try the live demo below!

🔢 int Integer

Whole numbers — positive, negative, or zero. No decimal point.

year = 2025
temp = -7
Fast math · Can grow infinitely large in Python!
💧 float Decimal

Numbers with a decimal point. Used for measurements, prices, percentages.

pi = 3.14159
tax = 0.13
0.1 + 0.2 ≠ 0.3 — floating point quirk!
💬 str String

Any text, wrapped in quotes. Letters, words, emojis — it's all a string!

name = "Alice"
msg = 'Hi! 👋'
Strings can be concatenated: "Hi" + " Alice"
✅ bool Boolean

Only two possible values: True or False. The on/off switch of programming!

logged_in = True
is_admin = False
Powers all if statements and logic!

What is a Variable? 🤔

In programming, a variable is a named memory storage box. You can put things inside it, change them later, or read what's stored — all by using its label.

age = 12
name = "Alice"

The Break Down:

  • age — the Label (Variable Name)
  • = — the Assignment Operator
  • 12 — the Value stored inside
Naming Rules: use snake_case, no spaces, no starting with a number!

Instead of remembering messy computer addresses like 0x7ffd4a, we just talk to boxes by their labels!

Variables Can Vary! 🔄

Variables aren't locked forever. You can throw out an old value and replace it with a fresh one instantly.

score = 0
score = 10
score = score + 5
